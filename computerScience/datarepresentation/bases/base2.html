Base 2

Base 2, otherwise known as Binary, is an incredibly easy way to write numbers, because we only have two numbers to choose from. Instead of using all ten numbers from 0 - 9 like in Base 10, we only use two numbers, 0 and 1 in Base 2. (Hopefully you can see why Base 10 is called Base 10 and Base 2 is called Base 2 now.)
